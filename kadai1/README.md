## リポジトリの概要
month_days
- 西暦とを入力するとその月が何日まであるかを教えてくれる


### リポジトリの使用方法
- *GitHub* から *robosys2024* のリポジトリをクローン  
`$ git clone https://github.com/SoichiroS1066/robosys2024/`  
- *kadai1* のディレクトリへ移動  
`$ cd ~/robosys2024/kadai1/`
